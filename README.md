# Funcionalidades

## Conexões
/connections
- Rotas para listar o total de conexões realizadas;
- Rota para criar uma nova conexão;

## Aulas
/classes
- Rota para criar uma aula;
- Rota para listar aulas;
  - Filtrar por matéria, dia da semana e horário;
